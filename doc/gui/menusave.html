The save menu

This menu allows you to save your filtered data and half-life results. The saving can also be performed via buttons in the corresponding menu panels.

Save (filtered) data

Here you can save your data before or after filtering to a chosen output file. Details on the file format can be found under File formats.

Save calculated values

This menu lets you save either calculated half-life, ratio or both values after half-life calculation for all of your half-lives combined. If you only want to save one calculation or want to save it with attributes like e.g. gene name, the menu provides the menu point Half-lives with attributes. For details on the file formats see File formats.
