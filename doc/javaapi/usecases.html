Use cases

This page provides an overview over the functionality of the different parts of the HALO software package. You can select between four different sample use cases, but we advise you to keep the proposed order, since the different parts are dependent on each other.

  1. Loading and filtering data
  2. Using normalization and half-life calculation methods
  3. Working with additional tools, e.g. evaluation

Sample Use Case 1: Loading and filtering data

Show use case

In this first use case you are shown how you can load and filter your data with HALO.

What will be done?
  • Loading expression data
  • Loading additional attributes
  • Filtering data
  • Printing data to output

Sample Use Case 2: Using half-life calculation methods

Show use case

The second use case introduces you to the subject of normalization and its basic usage. From this use case you can also learn how to handle half-life calculation methods and how to manage the results.

What will be done?
  • Choosing of normalization method
  • Performing normalization
  • Plotting normalization results
  • Calculation of half-lives
  • Plotting of half-lives
  • Printing of half-lives to a file
  • Calculate median half-life
  • Filtering with probeset quality scores

Sample Use Case 3: Working with additional tools, e.g. evaluation

Show use case

The last use case demonstrates the usage of two additional features of HALO, evaluation of data quality based on protein sequences and calculation of the median half-life.

What will be done?
  • Sequence-based evaluation
  • Calculation of median half-life
