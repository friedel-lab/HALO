Library overview


Here you can find an overview over the package structure of the HALO API.

halo.data
Contains implementations of data loading, management and filtering.

halo.normalization
Different implementations of normalization methods can be found in this package.

halo.halflife
This package contains implementations of half-life calculation methods.

halo.userinterface
Not documented here: Package containing implementations of graphical user interface, command line tools and related tools.

halo.examples
Implementations of example use cases. See use cases.

halo.tools
Contains helper classes.
