Installation guide

This section provides a quick guide on how to install and get HALO running on your system.

Extracting HALO

The HALO package can be downloaded from here. It is provided as both a gzipped tar and a zip archive.
  • halo-X.Y.tar.gz
  • halo-X.Y.zip
Where X.Y is the version number of the HALO package you downloaded.

Linux

You can use a graphical tool for the unpacking of the archive into a chosen directory, or simply extract it from the command line with the following command:

tar xvzf halo-X.Y.tar.gz
or
unzip halo-X.Y.zip

Note that this command will create a sub-directory halo-X.Y in the current working directory.

Windows

Use any archiving tool (e.g. 7-Zip) to unpack one of the archives (.zip or .tar.gz) into a directory of your choice.

Starting HALO
